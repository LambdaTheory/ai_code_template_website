## 文件说明

本文档由 `generate_structure.py` 脚本自动生成。

- 使用 `python generate_structure.py` 生成目录结构
- 使用 `python generate_structure.py --add <文件路径> "<描述>"` 添加文件描述
- 使用 `python generate_structure.py --missing` 查看缺少描述的文件
